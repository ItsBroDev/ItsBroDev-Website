Quick access to your favorite sites

If you use DuckDuckGo, the searchbar will also autocomplete !bangs, which let you quickly search on various websites. Additionally, ColdFOX includes another set of shortcuts that let you quickly access things within the browser. They'll always work regardless of what search engine you choose.

Browser Commands

!settings
Open the settings page
!back
Navigate backwards
!forward
Navigate forwards
!screenshot
Take a screenshot of the current tab
!clearhistory
Delete all history (excluding bookmarks)

Task Management

!task
Switch to a task
!task 1, !task shopping
!newtask
Create a new task
!newtask shopping
!closetask
Closes a task
!closetask shopping
!nametask
Renames the current task
!movetotask
Move the current tab to a task
!movetotask 1

Bookmarks & History

!bookmarks
View or search bookmarks
!bookmarks recipes
!importbookmarks
Import bookmarks from HTML file
!exportbookmarks
Export bookmarks to HTML file
!addbookmark
Bookmark the current page (optionally with tags)
!addbookmark tag1 tag2
!history
View a list of the pages you have visited recently

Content Blocking

!enableblocking
Enable content blocking on this site
!disableblocking
Disable content blocking on this site
